Table Cheat Sheet

HTML Table Elements

Name Tag Description
Table <table> represents tabular data â€” that is, information presented in a two-dimensional table comprised of rows and columns of cells containing data.
Table Body <tbody> encapsulates a set of table rows (<tr> elements), indicating that they comprise the body of the table (<table>).
Table Data <td> defines a cell of a table that contains data. It participates in the table model.
Table Footer <tfoot> defines a set of rows summarizing the columns of the table.
Table Header <th> defines a cell as the header of a group of table cells. The exact nature of this group is defined by the 'scope' and 'headers' attributes.
Table Head <thead> defines a set of rows defining the head of the columns of the table.
Table Row <tr> defines a row of cells in a table. The row's cells can then be established using a mix of <td> (data cell) and <th> (header cell) elements.
Column <col> defines a column within a table and is used for defining common semantics on all common cells. It is generally found within a <colgroup> element.
Column Group <colgroup> defines a group of columns within a table.

HTML Table Attributes

Name Tag Description
Column Span <colspan> Defines the number of columns a table cell should span over.
Row Span <rowspan> Defines the number of rows a table cell should span over.
